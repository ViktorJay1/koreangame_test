# koreangame_test
한국어학습게임 테스트 빌드
PC와 모바일에서 작동하는 “한국어 모음 소리-글자 매칭 게임” Level 1을 구현
단모음 10개 + 복합모음 9개 = 총 19개

## 로컬 실행
1. npm 스크립트로 실행
   ```bash
   npm run dev
   ```
2. 브라우저에서 열기
   ```
   http://localhost:8000
   ```

## 간단 확인
- 처음 로드 시 모음 소리가 재생되고, 다시 듣기 버튼으로 동일한 소리를 확인합니다.
- 3개의 버튼이 항상 보이며 클릭 시 해당 모음 소리가 재생됩니다.
